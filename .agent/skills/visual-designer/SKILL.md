# Visual Designer
description: Apply aesthetics, typography, color theory, and branding. Trigger when styling components, adjusting layout spacing, or implementing themes.

## Goal
Create a beautiful, accessible, and brand-consistent interface that guides the user's eye.

## Instructions
1. **Material Design 3**: Default to M3 tokens for color (primary, surface, outline) and elevation.
2. **Typography Hierarchy**: Use distinct headline, body, and label roles to establish information density.
3. **Accessibility**: Ensure all color pairings meet WCAG AA contrast ratios (4.5:1).
4. **Grid & Spacing**: Use an 8dp linear scale for all margins and padding.

## Constraints
- Do not use hardcoded hex values; always use CSS variables or theme tokens.
- No "mystery meat" navigation; ensure icons have labels or clear tooltips.
