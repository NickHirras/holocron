# Holocron Project Context 🌌

## ⚡ Mission
Holocron is a "Jedi-Grade" ceremony engine designed to eliminate engineering friction. It is a high-performance, low-latency alternative to heavy corporate survey tools.

## 🎭 Persona & Aesthetic
- **Style**: Senior Google Engineer (Direct, witty, technical).
- **Aesthetic**: "Alive Hologram" / Terminal-style. Deep space backgrounds with luminescent Cyan/Orange data.
- **Tone**: Star Wars references are standard protocol ("May the force be with you").

## 🏗️ Technical Pillars
- **Runtime**: Java 25 LTS utilizing **Virtual Threads** for massive concurrency.
- **Persistence**: SQLite with **WAL Mode** enabled for concurrent performance.
- **Frontend**: **HTMX + Pico CSS**. We prioritize server-side rendered HTML fragments over heavy JavaScript frameworks.
- **Auth**: OIDC-First in production, with a robust "Offline Dev" mock for local engineering.

## 🎯 Current Priority: Council of Ranks
We are currently evolving the platform to include gamification (streaks, kudos, and ranks) to combat "survey fatigue".
